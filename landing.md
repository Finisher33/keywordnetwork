# Gemini Canvas Landing Page Generation Prompt

아래 내용을 바탕으로 현대적이고 신뢰감 있는 디자인의 랜딩페이지를 생성해줘.

## [앱 정보]
- **앱 이름**: be Giver be Taker
- **한 줄 소개**: 리더의 나눔과 배움이 연결되는 지식 네트워크
- **타깃 사용자**: 기업 교육 과정 참가 리더 및 사내 지식 공유를 원하는 임직원
- **디자인 톤**: 전문적인(Professional), 신뢰감 있는(Trustworthy), 현대적인(Modern)

## [핵심 기능]
1. **키워드 기반 매칭**: Giver(줄 수 있는 것)와 Taker(배우고 싶은 것)를 연결하는 스마트 네트워크 추천
2. **티타임 요청**: 관심 있는 리더에게 즉시 대화를 제안하고 일정을 잡는 다이렉트 네트워킹
3. **학습 인사이트 공유**: 교육 세션별 핵심 배움을 동료들과 실시간으로 나누고 공감하는 피드
4. **네트워크 리포트**: 나의 연결 현황과 추천 네트워크를 한눈에 정리한 전문 PDF 리포트 발행
*그 외에도 과정/세션 관리, 근무지 기반 동료 추천, 데모 모드 체험 기능을 제공합니다.*

## [사용 방법]
1. **프로필 등록**: 소속 정보와 함께 나의 관심 키워드(Giver/Taker)를 입력합니다.
2. **네트워크 탐색**: AI가 추천하는 동료 리더들의 프로필과 학습 인사이트를 확인합니다.
3. **티타임 제안**: 더 깊은 교류가 필요한 리더에게 메시지와 함께 티타임을 요청합니다.

## [결과물 스펙]
- **기술 스택**: HTML + CSS + JS 한 파일 구성, Tailwind CSS CDN 사용
- **언어**: 한국어
- **디자인**: 모바일 우선 반응형 레이아웃, 외부 이미지 없이 이모지(Emoji)와 도형(SVG/CSS)으로 시각 요소 대체
- **섹션 구성**:
    1) **Hero**: 앱 이름, 한 줄 소개, "시작하기" 버튼 (강렬하고 깔끔한 첫인상)
    2) **기능 소개**: 위 4가지 핵심 기능을 카드 형태로 배치
    3) **사용 방법**: 1-2-3 단계별 설명 섹션
    4) **CTA 푸터**: "지금 써보기" 버튼과 제작자 정보

## [링크 설정]
- 코드 최상단에 `const APP_URL = "https://be-giver-be-taker.vercel.app";` 상수를 선언할 것
- Hero의 "시작하기" 버튼과 푸터의 "지금 써보기" 버튼은 모두 `APP_URL`을 참조할 것
- 모든 버튼 클릭 시 새 탭에서 열리도록 설정 (`target="_blank" rel="noopener"`)

- 만든 사람: 한상아 / 인재개발원
